pyHTTP2
=======

Python implementation of HTTP2 [draft 16](http://tools.ietf.org/html/draft-ietf-httpbis-http2-16 "draft 16")

## Simple usage
```
python serv_test.py <server ip> <server port>
```
```
python cli_test.py <server ip> <server port>
```
